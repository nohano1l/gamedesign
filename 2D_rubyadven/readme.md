# 2D Ruby's Adventure
### 操作方式
`方向鍵`可以控制Ruby移動，`X鍵`可以與NPC對話，`空白鍵`可以發射發條修理壞掉的機器人，但要注意發條數，可以撿拾地上的發條包補充發條數，除了注意壞掉的機器人外，也要小心地上的尖刺，草莓可以幫Ruby回血，開始你的旅行吧!
